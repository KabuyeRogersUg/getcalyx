Versatile

Calyx has got a wide range of modules and builtin functionality that adds os utilities to your project.

Calyx interpreter is a programming language just as Python and Ruby. It boosts with its strong dynamic typing without any boundaries. The language is Object Oriented and uses modulizing system to split a project in different files and folders.

Keywords here are not much different from those of C, Rust and Python. They are easy to memorize and short to reduce source code size.

The language sincerely is not stable yet, it is missing alot of functionality which is still being developed. Never the less, redownload the executable whenever you visit the website and expect some feature to be added.
